This still is one of my favorite projects while working at State Farm, this site uses Bootstrap, and Flexbox with heavy backend influence from Javascript and JQuery to load in different sections, load the list of active members, and load the terrirtories that are active. Tech Astra

This was my second big project at State Farm, this site is for girls in grades 6-12 to learn about coding, computer science based projects, and how to work as a team! I was a big fan of the message of this site, so I put a lot of hardwork and care into this site. It had a working clock set to the day the event begins, optimized for different timezones. We used Bootstrap with Flexbox to make the layout much more responsive, and SCSS with Javascript to make sure that the backend was synced with the frontend. This way the site was easy to read, and easy to understand for anyone who visited!
